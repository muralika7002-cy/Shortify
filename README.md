# Shortify

> Compress. Share. Track.

Shortify is a modern URL shortening platform that transforms long URLs into clean, memorable, and shareable links. Users can create custom aliases, manage their links from a personal dashboard, track click counts, and generate QR codes for quick sharing.

---

## Overview

Shortify provides a simple and efficient way to shorten URLs while maintaining a clean user experience. It is designed for students, professionals, developers, and anyone who frequently shares links on resumes, portfolios, social media profiles, emails, or business cards.

## How It Works

1. Create an account.
2. Log in to the platform.
3. Enter a long URL.
4. Optionally provide a custom alias.
5. Generate a short URL.
6. Receive a QR code for the generated link.
7. Manage links from the dashboard.
8. Track clicks through the built-in counter.
